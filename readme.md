## request haistyle

```bash
http GET http://localhost:3000/api/v1/hairstyles
```

## Post hairstyles

```bash
http POST :3000/api/v1/hairstyles name="Classic Cut" description="A timeless classic look" image_url="http://example.com/classic.jpg"

```

## Update hairstyles

```bash
http PUT :3000/api/v1/hairstyles/2 name="Under Cut bro" description="A timeless classic look" image_url="http://example.com/classic.jpg"
```

## Delete hairstyles

```bash
 http DELETE http://localhost:3000/api/v1/hairstyles/2
```
